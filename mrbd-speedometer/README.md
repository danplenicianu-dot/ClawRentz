# MRBD Speedometer

Minimal GPS speedometer web app for Meta Ray-Ban Display glasses.

## Built for MRBD constraints

- 600 x 600 viewport
- black page background for transparent additive display
- bright high-contrast speed readout
- one focusable D-pad action
- Enter / tap toggles GPS
- Escape stops GPS
- no external libraries or fonts

## Files

- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
